# OpenClash Assistant 中文说明

## 项目简介

`OpenClash Assistant` 是一个面向 iStoreOS / OpenWrt 的 LuCI 辅助插件，用来把 OpenClash 常见的排障、访问检查和辅助操作整理成一个更直观的页面。

当前版本主要提供：

- `访问检查`
  自动检查流媒体与 AI 目标的连接状态、延迟和返回结果
- `AI 检测`
  汇总 ChatGPT / OpenAI、Claude、Gemini 的访问状态
- `DNS 工具`
  提供 `Flush DNS` 能力
- `自动切换`
  辅助查看并应用 OpenClash 自动切换相关设置
- `订阅转换`
  辅助生成和写入订阅转换配置

## 适用环境

- iStoreOS
- OpenWrt
- 已安装 LuCI
- 已安装或正在使用 OpenClash

## 安装方式

### 一键安装命令

```sh
cd /tmp && curl -L -o openclash-assistant-istoreos-v0.1.0-r1.run https://github.com/Zhemuy-1a1a1/openclash-assistant-openwrt/releases/download/v0.1.0/openclash-assistant-istoreos-v0.1.0-r1.run && chmod +x openclash-assistant-istoreos-v0.1.0-r1.run && sh openclash-assistant-istoreos-v0.1.0-r1.run
```

### 方式一：使用 `.run` 安装包

```sh
chmod +x openclash-assistant-istoreos-v0.1.0-r1.run
./openclash-assistant-istoreos-v0.1.0-r1.run
```

### 方式二：使用 `ipk` 包

```sh
opkg install luci-app-openclash-assistant_0.1.0-1_all.ipk
```

如果系统提示依赖问题，请先确保 LuCI 相关依赖已安装。

## 卸载方式

如果你使用 `.run` 安装：

```sh
./openclash-assistant-istoreos-v0.1.0-r1.run --uninstall
```

如果你使用 `ipk` 安装：

```sh
opkg remove luci-app-openclash-assistant
```

## 页面入口

安装完成后，在 LuCI 里进入：

`服务` -> `OpenClash Assistant`

## 主要功能说明

### 1. 访问检查

- 默认对全部流媒体与 AI 目标自动执行访问检查
- 进入标签页后会自动触发
- 检查过程中只局部刷新结果，不会整页反复刷新
- 正常项会以更简洁的卡片展示

### 2. AI 检测

- 提供 ChatGPT / OpenAI、Claude、Gemini 的访问摘要
- 详细结果统一在 `访问检查` 标签里查看

### 3. DNS 工具

- 提供 `Flush DNS`
- 会优先重启 `dnsmasq`
- 如果系统启用了 `smartdns`，也会一起重启
