# OpenClash Assistant v0.1.0

OpenClash Assistant for iStoreOS / OpenWrt.

## Highlights

- Unified `访问检查` panel for streaming and AI connectivity checks
- Default full-target auto checks with local card refresh
- Separate `DNS 工具` tab with `Flush DNS`
- `自动切换` and `订阅转换` tabs kept available for OpenClash workflows
- iStoreOS / OpenWrt one-file installer included

## Included Assets

- `openclash-assistant-istoreos-v0.1.0-r1.run`
- `openclash-assistant-istoreos-v0.1.0-r1.run.sha256`
- `openclash-assistant-istoreos-v0.1.0-r1-release.tar.gz`

## Install

```sh
chmod +x openclash-assistant-istoreos-v0.1.0-r1.run
./openclash-assistant-istoreos-v0.1.0-r1.run
```

## Uninstall

```sh
./openclash-assistant-istoreos-v0.1.0-r1.run --uninstall
```
